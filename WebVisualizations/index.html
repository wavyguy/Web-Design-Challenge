Summary of Latitude Analysis Results (Effects)
By: Trey Bailey (SMU Data Science - 2020)





For this assignment, we analyzed weather trends based on distance to the equator. We used Python to visualize the weather of over 500 cities across the world, studying the distance from the equator and the effect on the weather in that given city.

Besides the simple Python library, we utilized the OpenWeatherMap API and a little common sense to create a representative model of weather across world cities. Hope you enjoy this web design of our project!
